# 开发路线图 (Roadmap)

## 当前状态: v1.0.0 (首次发布)

**已完成**: 从剧本到视频的完整工作流

---

## 短期目标 (v1.1.0) - 1-2周

### 高优先级
- [ ] **视频状态查询修复**
  - 找到稳定的状态查询端点
  - 或实现Webhook回调机制
  
- [ ] **批量操作优化**
  - 批量审核角色/场景
  - 批量重新生成参考图
  - 批量重新生成分镜

- [ ] **UI/UX改进**
  - 添加加载状态指示器
  - 错误提示优化
  - 操作确认对话框

### 中优先级
- [ ] **项目导出功能**
  - 视频序列合并
  - 项目打包下载
  - 时间轴预览

- [ ] **提示词模板库**
  - 预设提示词模板
  - 用户自定义模板
  - 模板导入/导出

---

## 中期目标 (v1.2.0) - 1-2月

### 新功能
- [ ] **多视频提供商支持**
  - 可灵 (Kling) 集成
  - Runway ML 集成
  - Pika Labs 集成
  - 统一配置界面

- [ ] **角色一致性优化**
  - 角色参考图权重调整
  - 跨场景角色一致性检查
  - 角色特征提取优化

- [ ] **高级分镜控制**
  - 关键帧插值
  - 镜头运动路径
  - 多角度分镜预览

### 性能优化
- [ ] **图片缓存优化**
  - 智能缓存策略
  - CDN集成
  
- [ ] **队列优化**
  - 优先级队列
  - 任务依赖管理

---

## 长期目标 (v2.0.0) - 3-6月

### 架构升级
- [ ] **多用户支持**
  - 用户认证系统
  - 权限管理
  - 项目隔离

- [ ] **云存储集成**
  - AWS S3 / 阿里云OSS
  - 云端项目同步
  - 团队协作

- [ ] **Webhook系统**
  - 外部系统集成
  - 自动化工作流
  - 第三方插件支持

### AI能力提升
- [ ] **智能剧本分析**
  - 情感分析
  - 节奏建议
  - 自动分镜优化

- [ ] **风格迁移**
  - 参考图风格提取
  - 跨风格迁移
  - 风格一致性保证

- [ ] **语音合成集成**
  - 角色配音
  - 口型同步
  - 多语言支持
